Some question & answer about DOM

  • What is dom and what is the purpose of DOM?

    • Document object model is known as DOM. Web documents are accessed through this programming interface.

    Purpose of DOM:

    1. Each document element is represented as a node in the tree by the DOM.
    2. The text, structure, and style of a document can all be dynamically accessed and changed by programs and scripts because to this feature.
  • How will you select HTML elements using DOM? Name the DOM methods.

    • To select HTML elements using DOM, you can use some DOM methods in JavaScript, such as

    1. getElementById()
    2. getElementsByClassName()
    3. getElementsByTagName()
    4. querySelector()
    5. querySelectorAll()
  • What is event bubble?

    • When an event, such as a click, occurs on a web page element, it is first handled by that element before moving up the parent elements in the HTML hierarchy until it reaches the top level of the page. This process is known as event bubbling.

  • What is a callback function and why will you use it?

    • A callback function allows you to assign a certain task to another function and have it perform asynchronously. The main function will keep running, and the callback function will be called when the task is finished or the event takes place.
